# 行为准则

## 我们的承诺

作为项目成员、贡献者和领导者，我们承诺让参与我们社区的每个人都不受骚扰，无论其年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、种姓、肤色、宗教信仰、性取向或技术偏好如何。

## 我们的标准

有助于营造积极环境的行为包括：

- 使用友善和包容的语言
- 尊重不同的观点和经验
- 虚心接受建设性批评
- 关注社区整体利益
- 对其他成员表达同理心

不可接受的行为包括：

- 使用带有性暗示的言语或图像
- 挑衅、侮辱或贬损性评论
- 公开或私下骚扰
- 未经同意发布他人私人信息
- 其他可被合理认定为不专业或破坏性的行为

## 执行责任

项目维护者有责任明确行为准则标准，并对任何他们认为不适当、威胁、冒犯或有害的行为采取适当且公正的纠正措施。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共场所代表项目或社区时的行为。

## 报告与处理

如遇到滥用、骚扰或其他不可接受的行为，请通过邮件 conduct@autoship.dev 联系项目团队。我们将尽快审查并调查所有投诉，并以适当方式回应。

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版。
