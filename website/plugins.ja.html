プラグイン

セキュリティ、デリバリー、Issue 追跡などのコミュニティ・検証済みプラグインで AutoShip を拡張しましょう。

レジストリを見る プラグインを作る

注目のプラグイン

security-scan

毎回のコミット前に bandit、gitleaks、osv-scanner を実行。

autoship plugin install security-scan

docker-ship

--target docker でアップロードする際に Docker イメージをビルド・プッシュ。

autoship plugin install docker-ship

web-search

検証失敗時に Web を検索してエラーの文脈を取得。

autoship plugin install web-search

信頼レベル

builtin

AutoShip に同梱。追加の検証は不要。

verified

AutoShip チームがレビューし、チェックサムまたは署名付き。

community

コミュニティメンバーが公開。インストール前に警告を表示。

untrusted

ローカルパスや不明なソースからインストール。注意して使用。
