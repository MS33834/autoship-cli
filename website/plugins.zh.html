插件

通过社区和已验证的插件扩展 AutoShip，支持安全扫描、交付、问题追踪等功能。

浏览注册表 开发插件

精选插件

security-scan

每次提交前运行 bandit、gitleaks 和 osv-scanner。

autoship plugin install security-scan

docker-ship

使用 --target docker 上传时构建并推送 Docker 镜像。

autoship plugin install docker-ship

web-search

验证失败时搜索网络获取错误上下文。

autoship plugin install web-search

信任等级

builtin

随 AutoShip 一起发布，无需额外验证。

verified

由 AutoShip 团队审核，并带有校验和或签名。

community

由社区成员发布，安装前会显示警告。

untrusted

从本地路径或未知来源安装，请谨慎使用。
