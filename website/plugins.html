Plugins

Extend AutoShip with community and verified plugins for security, shipping, issue tracking, and more.

Browse Registry Build a Plugin

Featured plugins

security-scan

Run bandit, gitleaks, and osv-scanner before every commit.

autoship plugin install security-scan

docker-ship

Build and push Docker images when uploading with --target docker.

autoship plugin install docker-ship

web-search

Search the web for error context when verification fails.

autoship plugin install web-search

Trust levels

builtin

Ships with AutoShip. No extra verification needed.

verified

Reviewed by the AutoShip team and signed with a checksum or signature.

community

Published by community members. We show a warning before installation.

untrusted

Installed from a local path or unknown source. Use with caution.
