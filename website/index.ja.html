AI でコードをデリバリーし、ローカルに留まる。

AutoShip はローカル優先、AI 支援のツールキットです。コードベースをクラウドに送信することなく、クリーンアップ、検証、コミット、デリバリーを自動化します。

pip install autoship はじめる

機能

ローカル AI

Ollama、LM Studio などのローカルモデルバックエンドに対応。コードはあなたのマシンに留まります。

スマートコミット

ステージ済みの変更から規約に沿ったコミットメッセージを生成し、適用前に確認できます。

検証可能なツールチェーン

設定可能なツールパスと SHA256 チェックにより、autoship clean、verify、upload を実行。

プラグインエコシステム

セキュリティスキャン、Docker デリバリー、Web 検索など、プラグインで AutoShip を拡張。

エンタープライズ対応

チーム設定、構造化された監査ログ、SIEM 転送、多言語 CLI をサポート。

安全性優先

エディタ許可リスト、コマンド許可リスト、環境変数の上書き、Telemetry エンドポイント検証。

すぐに使えるワークフロー

# 1. 初期化
autoship init

# 2. クリーンアップとフォーマット
autoship clean

# 3. 検証
autoship verify pytest

# 4. AI 生成のコミットメッセージ
autoship commit

# 5. アーティファクトのアップロード
autoship upload --target pypi

拡張のために設計

AutoShip SDK を使えば、 数分で独自のプラグインを構築・公開できます。 

pip install autoship-sdk
autoship-sdk create my-plugin

プラグインレジストリをブラウズして、コミュニティや検証済みプラグインを見つけましょう。
