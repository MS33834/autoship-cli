用 AI 交付代码，保持本地优先。

AutoShip 是一款本地优先、AI 辅助的工具包，帮助你在不将代码库上传到云端的情况下完成清理、验证、提交和交付。

pip install autoship 开始使用

功能特性

本地 AI

兼容 Ollama、LM Studio 等本地模型后端，代码始终保留在你的机器上。

智能提交

根据已暂存改动生成符合规范的提交信息，并在应用前供你审阅。

可信工具链

通过可配置的工具路径与 SHA256 校验运行 autoship clean、verify 和 upload。

插件生态

通过插件扩展 AutoShip，支持安全扫描、Docker 交付、Web 搜索等功能。

企业就绪

支持团队配置、结构化审计日志、SIEM 转发和多语言 CLI。

安全第一

编辑器白名单、命令白名单、环境变量覆盖与遥测端点校验。

开箱即用的工作流

# 1. 初始化
autoship init

# 2. 清理并格式化
autoship clean

# 3. 验证
autoship verify pytest

# 4. AI 生成提交信息
autoship commit

# 5. 上传产物
autoship upload --target pypi

为扩展而生

使用 AutoShip SDK 在几分钟内构建并发布你自己的插件。 

pip install autoship-sdk
autoship-sdk create my-plugin

浏览 插件注册表，发现社区和已验证插件。
